# helm-charts

## Package the Helm Chart

```
$ helm package ../k8s-examples/helm-example
$ helm package ../k8s-examples/helm-example2
```

## Update the Chart Index

```
$ helm repo index --url https://cg5labs.github.io/helm-charts .
```
